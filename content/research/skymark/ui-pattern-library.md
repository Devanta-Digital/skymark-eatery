# UI pattern library — evidence-backed patterns for Skymark

## Layout rhythm

- **Alternating surfaces:** light cream → deep espresso → muted sand → deep again. Never more than two adjacent sections in the same tone.
- **Vertical breathing:** marketing pages that feel “expensive” use **more space between sections** than inside cards.
- **One asymmetry per homepage:** e.g. oversized category tile or split grid with unequal columns.

## Typography

- **One serif display** for H1/H2 emotional beats only.
- **One grotesk / clean sans** for UI, meta, and body at smaller sizes.
- **Kickers:** small uppercase, wide tracking, single accent colour only.

## CTA strategy

- **Max two visible primaries** in hero; tertiary actions become text links.
- **Catering** uses the same filled CTA language as “order” but must not compete in the same visual band without hierarchy.

## Motion

- **Fade-up + stagger** on section entry; **no** layout-affecting springs.
- Hover: **2–4px lift** on tiles max.

## Mobile conversion

- Sticky bottom bar: **two actions max** (Menu + Catering quote).
- Tap targets ≥ 44px height on mobile CTAs.
