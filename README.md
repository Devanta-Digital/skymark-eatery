# Skymark Eatery by Caffè É Pranzo

  A production-ready Italian restaurant platform built with React, Node.js, and PostgreSQL.

  ## Features
  - Online menu & ordering with Stripe payments
  - Real-time kitchen display (SSE)
  - Admin dashboard (orders, menu, specials, finance, settings)
  - AI menu editor with AI image generation
  - Catering request system
  - Staff & role-based access control

  ## Stack
  - **Frontend**: React + Vite + TailwindCSS + shadcn/ui
  - **Backend**: Express + Drizzle ORM + PostgreSQL
  - **AI**: Anthropic Claude (menu editing + image prompts)
  - **Payments**: Stripe
